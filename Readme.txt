Steps to run the app.
Step 1: Unzip the folder.

Step 2: Install Python 3.11 (if not installed)

Step 3: open the unziped folder in command prompt(cmd) or any code editor terminal.

Step 4: Activate the virtual envirnment with 
	command 'venv/Scripts/activate' and press enter.

Step 5: Install the required packages.
	type command 'pip install flask' and press enter.
	type command 'pip install yfinance' and press enter.
	type command 'pip install matplotlib' and press enter.

Step 8: Run the app with
	command 'py init.py' and press enter.

Step 9: Copy the URL and Paste in web Browser.
